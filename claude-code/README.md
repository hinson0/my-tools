# CLAUDE.local.md

🌐 English | [简体中文](./README.zh-CN.md)

A reusable personal Claude Code configuration file (`.CLAUDE.local.md`) that can be symlinked into any project for consistent behavior across repositories.

## Installation

```bash
ln -s ~/my-tools/claude-code/.CLAUDE.local.md ~/your-project/CLAUDE.local.md
```
